# Multiline-editor
Designed and implemented a stateful command-line text editor in Python inspired by Vim. Engineered a command interpreter with support for multi-mode operations, regex-driven navigation, and persistent undo/redo using snapshot-based history management.
The editor includes an interactive help feature, triggered by typing "?". This displays a full list of available commands and their usage, making it easy for users to explore all functionality without reading the source code.
